The Solana SDK targets the deployed Arcane program bundled with the SDK IDL.

```text
2EyitVDBV38LSeGDKQuZWHMCmebMdGeVFgHC5LYxepLG
```

Confirm the target program id with Arcane before production deployment.

## SDK responsibilities

The SDK includes:

- Anchor program helpers and PDA derivation.
- UTXO keypair and commitment/nullifier models.
- Private SOL deposit, transfer, withdrawal, and UTXO consolidation flows.
- Indexer, relayer, Light RPC, Merkle proof, and prover utilities.
- Token metadata helpers for SOL and dynamic private assets.

## Application responsibilities

Your application provides:

- Product backend routes.
- Session handling.
- Database persistence.
- Backend-managed signing.
- Product reconciliation.
- User interface.

These application-specific pieces are not part of the SDK package.
