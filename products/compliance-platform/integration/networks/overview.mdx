The Compliance Platform works with compatible confidential applications and
assets on Solana, Stellar, and Fhenix.

| Network | Compatible privacy technology | Current integration documentation |
| --- | --- | --- |
| **Solana** | Arcane Privacy Layer and Token-2022 Confidential Transfer | Service API Asset flow for the Solana SDP integration |
| **Stellar** | Arcane Privacy Layer | Compatibility overview |
| **Fhenix** | Fhenix confidential tokens using FHE | Compatibility overview |

The current Service API contract is specific to the Solana confidential-token
flow. It should not be copied into Stellar or Fhenix documentation until the
request fields, network identifiers, and ingestion behavior for those
integrations are verified.
