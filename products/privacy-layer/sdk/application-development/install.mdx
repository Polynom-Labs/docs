---
title: Install
description: Install the Stellar privacy client and Redux state adapter in your application
---

Add `@arcanetech/privacy-sdk-stellar` and `@arcanetech/privacy-sdk-state-redux` to your React application. The Redux adapter also requires `@reduxjs/toolkit` as a peer dependency.

<CodeGroup>

```bash npm
npm install @arcanetech/privacy-sdk-stellar @arcanetech/privacy-sdk-state-redux @reduxjs/toolkit react-redux
```

```bash yarn
yarn add @arcanetech/privacy-sdk-stellar @arcanetech/privacy-sdk-state-redux @reduxjs/toolkit react-redux
```

```bash pnpm
pnpm add @arcanetech/privacy-sdk-stellar @arcanetech/privacy-sdk-state-redux @reduxjs/toolkit react-redux
```

</CodeGroup>

Next, wire the client in your app shell. See [Setup](./setup).
